O VITE não reconhece rotas. ou seja, para transitar entre uma página e outra, precisa ensinar pra ele. precisa instalar o pacote.

TIPOS DE ROTAS 

- Estáticas
um endereço fixo, previsível. acessar diretamente no código. são caminhos pre-definidos. ex: home, contato, sobre...
essa rota não muda com parametro. o caminho é sempre o mesmo /sobre, /contato, /faq.
precisa criar roteamento.

=> é necessário instalar o REACT-ROUTER-DOM
ele é uma biblioteca para gerenciar a navegação em react.

vamos trabalhar com: 

- BrowserRouter
- Routes
- Route
- Link (links agora externos usa-se a tag <a></a>, mas para links internos não)

para usar o pacotes, precisa colocar o comando: npm install react-router-dom

vá em App.tsx e dentro de <></> colocar o BrowserRouter.
depois, precisa mostrar o conjunto de rotas.
precisa importar com <Routes></Routes>.
depois, precisa especificar tudo: caminho e elemento.

      <Routes>
        <Route path="/" element={Home}></Route>
      </Routes>

      OBS: A / significa que vai entrar no home direto.

é necessário importar tambem o Route. o código vai ficar assim:

import { BrowserRouter, Route, Routes } from 'react-router-dom';
import './App.css';
import Footer from './components/Footer/Footer';
import Header from './components/Header/Header';
import Home from './pages/Home/Home';
import Sobre from './pages/Sobre/Sobre';
import Contato from './pages/Contato/Contato';

function App() {
  return (
    <>
      <BrowserRouter>
        <div className="min-h-screen flex flex-col">
          <Header />

          <Routes>
            <Route path="/" element={<Home />} />
            <Route path="/sobre" element={<Sobre />} />
            <Route path="/contato" element={<Contato />} />
          </Routes>

          <Footer />
        </div>
      </BrowserRouter>
    </>
  );
}

export default App;

precisamos criar a página de erro 404 - página não encontrada.
para criar é fácil, cria uma pasta dentro de "pages" chamanda NotFound, e um documento tsx tambem. é o mesmo raciocinio dos outros componentes.
para implementar, em Routes, colocar ela com /*.

vai ficar assim:

          <Routes>
            <Route path="/" element={<Home />} />
            <Route path="/sobre" element={<Sobre />} />
            <Route path="/contato" element={<Contato />} />
            <Route path="/*" element={<NotFound />} />
          </Routes>

para adicionar imagens existem dois tipos: armazenadas na pasta public e armazenadas em assets.

=> na pasta public:
todo mundo acessa. não precisa importar.
<img src="/heart.png" />


=> na pasta assets: 
somente o server, o node tem acesso. precisa importar.
import icon404 from '../../assets/img/404.png'

exemplo:

import icon404 from '../../assets/img/404.png'

const NotFound = () => {
    return(
        <>
        <main className="flex-1 container mx-auto p-6 flex flex-col items-center justify-center">
            <div className="grid grid-cols-1 gap-6">
                <h1 className="text-6xl font-semibold mb-2 text-center">Página não encontrada!</h1>
                <img src={icon404} />
            </div>
        </main>
        </>
    )
}
export default NotFound;
