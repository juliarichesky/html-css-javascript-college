VITE
=> oferece experiencia de desenvolvimento rapida, moderna e eficiente.
=> NÃO É FRAMEWORK
=> https://vite.dev/ -- documentação

PASSO A PASSO PARA USAR O VITE:

- abre o VS Code
- clica com botão direito e clica em "open in integrated terminal"
- abre o terminal e digita: npx create-vit@latest
- se pedir pra apertar o y, aperta e pode criar o nome do projeto.
CASO DE ERRO DE SEGURANÇA:
- abre o powershell como adm e execute o comando: Set-ExecutionPolicy RemoteSigned
- confirma a alteração digitando A no terminal e aperte enter
- feche o powershell
- execute de novo o powershell sem o modo adm
- coloque o comando: node --version
- se aparecer versão do node, foi corrigido o problema.

ele vai criar uma pasta com o nome que voce deu, e depois varias coisas dentro, nomes de arquivos, etc.
=> para abrir novamente o localhost, abra o terminal DENTRO DO PROJETO e digite: npm run dev

NPX => Node Package Manager: ele executa pacotes do NPM (Node.js) diretamente da linha de comando.

REGRAS
=> nome do projeto: sem nome composto, caractere especial, letras maiusculas, espaços, nao pode começar com numero e underline e hifen. somente letras minusculas, hifens (-) e underline (_)
