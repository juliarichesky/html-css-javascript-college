ECMAscript e JavaScript

- é uma linguagem derivada do C e do Java
- linguagem interpretada, nao compilada (clientSide)
- recomendado uso de ; em todo final de linha
- diferencia letras maiusculas e minusculas (case sensitive)
- comentarios feitos com // ou /* */
- nao é obrigado a declarar variavel
- variavel dinamicamente tipadas, ela aceita qualquer tipo de conteudo.
- var / let / const

=> var
variavel global, onde eu declarar o meu script todo vai conseguir enxergar a variavel. e nao importa onde eu declare, inicio, meio, final, nao importa pois o script vai conseguir captar.
(HOISTING)
recomendado declarar ela no inicio

=> let
variavel local, so consigo enxergar a variavel dentro da função, nao da para enxergar fora

=> const
nao pode haver alteração, é um valor imutavel


- há duas formas de trabalhar com script, interno e externo.
- interno: dentro de head com tag <script></script>
- algumas tags script, em alguns momentos, vamos escrever o script depois do body. depende muito da necessidade.
- google sugere para colocar o script no final ANTES do fechamento do html
- eventos sao gatilhos que vao disparar alguma  função
- função obrigatoriamente usa-se parenteses. ()
